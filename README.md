# ⚠️ **BETA**
 
***

## ⬇️ Installation

```npm install --global @turtle-todo/turtle.cli```

## 📕️ Documentation

```turtle add``` Add an item to your todo list

```turtle remove``` Remove an item

```turtle list``` Show all todo items

```turtle set``` Set the path to your todo.json
